# Multi-Line-Text-Processing-Tool

A Java application for processing multi-line text input with formatting, sorting, and manipulation capabilities.

## Features
- Accept 1-5 lines of text input
- Parse and format words with consistent spacing
- Truncate words to 10 characters
- Alphabetically sort words within each line
- Display formatted output with delimiters

## How to Run
1. Compile: `javac Project3.java`
2. Run: `java Project3` or `java Project3 <number_of_lines>`
3. Enter your lines of text when prompted

## Skills Demonstrated
- 2D array manipulation
- String formatting and truncation
- Sorting algorithms (bubble sort)
- Input validation
- Command-line arguments handling
